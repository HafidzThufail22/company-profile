# LPK Sadewa - Website Kursus Mengemudi

Website company profile untuk **LPK Sadewa**, Lembaga Pendidikan Kursus Mengemudi.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18+
- npm atau yarn

### Installation

1. Clone repository
```bash
git clone https://github.com/HafidzThufail22/company-profile.git
cd company-profile
```

2. Install dependencies
```bash
npm install
```

3. Jalankan development server
```bash
npm run dev
```

4. Buka [http://localhost:3000](http://localhost:3000) di browser
